# DevTools Pro

**DevTools Pro** is a comprehensive, offline-ready utility suite designed for developers. It aggregates over 15 essential tools into a single, modern, and responsive interface, allowing you to perform common tasks quickly without relying on multiple online services.

## üöÄ Features

DevTools Pro includes a wide range of utilities categorized for easy access:

*   **JSON Tools**:
    *   **Beautify/Minify**: Format complex JSON or reduce its size.
    *   **Validate**: Quickly check if your JSON string is valid.
*   **Base64 Guru**:
    *   **Advanced Encoding**: Convert text, files, hex strings, or URLs to Base64.
    *   **Decoding**: Decode Base64 back to text, images, audio, video, or PDF.
    *   **Data URI Support**: Easily generate `data:` URIs for embedding assets.
*   **Text Utilities**:
    *   Case conversion (Uppercase/Lowercase).
    *   Find & Replace.
    *   Regex Tester.
    *   Line sorting and statistics (word/char count).
*   **Security & Cryptography**:
    *   **Hash Generator**: SHA-256, SHA-1, MD5.
    *   **Password Strength Checker**.
    *   **JWT Decoder**: Parse and view JSON Web Tokens.
*   **API & Network**:
    *   **REST API Tester**: Send GET, POST, PUT, DELETE requests and view responses.
*   **Date & Time**:
    *   Unix Timestamp conversion.
    *   ISO-8601 date generation.
*   **Frontend Tools**:
    *   **CSS Utilities**: Minify and Beautify CSS.
    *   **Color Converter**: HEX <-> RGB.
    *   **Generators**: CSS Gradients and Box Shadows with live preview.
    *   **Font Preview**: Test different font families.
*   **Code Utilities**:
    *   Simple code minification and obfuscation.
*   **Database**:
    *   **SQL Query Builder**: Visually construct simple SQL queries.
*   **DevOps**:
    *   **Dockerfile Generator**: Create basic Dockerfile templates.
